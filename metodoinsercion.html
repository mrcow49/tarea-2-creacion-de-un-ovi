Metodo de Ordenamiento por Inserción


Para este método utilizaremos un vector auxiliar con el mismo tamaño que el vector base, de hecho, deberá ser una copia idéntica del vector base. El método consiste en recorrer el vector desde la posición 0 hasta el final y a medida que se encuentre un valor menor este se ubica en la posición que le corresponde comparándolo con los anteriores valores y los valores siguientes deberán desplazarse en el mismo orden que tienen hasta el momento.
